Detailed Description

Callback interface to deliver the "connection created" event to a receiving application.

The application supplies an implementation of this interface to LbmCtRcv#start.

Receiver application callback method, called by CT, to indicate that a new connection is created.

Parameters

Returns
Application-specific object to be associated with the connection, which is made available to the application when messages are delivered, and LbmCtRcvConnDeleteCallback#onRcvConnDelete. This allows per-connection state to be maintained.
